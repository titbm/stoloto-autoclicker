# Столото Автокликер v2.0

Расширение Chrome для автоматического поиска билетов Русского лото с заданными числами.

## Архитектура

Проект построен на принципах **DDD** и **Clean Architecture** с упором на **YAGNI**.

Подробнее: [src/ARCHITECTURE.md](src/ARCHITECTURE.md)

## Структура проекта

```
src/
├── domain/              # Бизнес-логика (Ticket, SearchCriteria, UserData)
├── usecases/            # Сценарии использования (SearchTickets)
├── adapters/            # Адаптеры к внешнему миру (WebsiteAdapter, ChromeAdapter)
├── ui/                  # Интерфейс (sidepanel)
├── shared/              # Общие модули (constants, messaging, errors)
├── background.js        # Service Worker
└── content.js           # Content Script

manifest.json            # Манифест расширения
```

## Установка

1. Клонировать репозиторий
2. Открыть Chrome → `chrome://extensions/`
3. Включить "Режим разработчика"
4. Нажать "Загрузить распакованное расширение"
5. Выбрать папку проекта

## Использование

1. Перейти на страницу [Русское лото](https://www.stoloto.ru/ruslotto/game?viewType=tickets)
2. Открыть боковую панель расширения
3. Ввести числа для поиска
4. Выбрать режим поиска
5. Нажать "Запустить поиск"

## Режимы поиска

- **Числа где угодно** - ищет билеты, где есть все указанные числа
- **Числа в одной половине** - все числа в верхней или нижней половине билета
- **Числа в одной строке** - все числа в одной строке билета

## Разработка

### Принципы

1. **DDD** - Domain-Driven Design
2. **Clean Architecture** - слоистая архитектура
3. **YAGNI** - не создаем код "на будущее"

### Правила

- За 1 раз редактируем только 1 файл
- Перед изменениями - обсуждаем
- Работаем мелкими шагами
